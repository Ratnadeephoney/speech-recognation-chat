          how to intigrate bootstrap with angular app
	|
          open cmd in project path:
	|
npm install --save bootstrap
	|
npm install @ng-bootstrap/ng-bootstrap
	|
npm install @popperjs/core --save
	|
          open angular.json file
	|
do changes :
	
"styles": [
              "node_modules/bootstrap/dist/css/bootstrap.min.css",//add this line
              "src/theme/variables.scss",
              "src/global.scss"
            ],
"scripts": [
              "node_modules/jquery/dist/jquery.min.js",//add this line
              "node_modules/bootstrap/dist/js/bootstrap.min.js"//add this line

            ],
	
             
          open app.module.ts file and do some changes:
	|
import { NgbModule } from '@ng-bootstrap/ng-bootstrap';

imports:[
	NgbModule,
            ]
